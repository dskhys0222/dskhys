# 要件定義：金額増減ダイアログ（Budget）

## 目的

トップページで現在の金額を表示し、`+` ボタンまたは `-` ボタン押下時にダイアログを表示して、ユーザーが「増減する金額（差分）」を入力できるようにする。

## 対象範囲

- クライアント：`packages/client/budget`

## ユーザーストーリー

- `+` を押して増額したい金額を入力し、現在金額に加算できる
- `-` を押して減額したい金額を入力し、現在金額から減算できる
- 入力をキャンセルした場合は金額が変わらない

## 機能要件

1. 現在金額と `+` / `-` ボタンを表示する
2. `+` ボタン押下で「増額」ダイアログを表示する
3. `-` ボタン押下で「減額」ダイアログを表示する
4. ダイアログには、増減する金額（差分）を入力するフィールドと、確定/キャンセル操作を提供する
5. 確定時、入力した差分を現在金額へ反映する（`+` は加算、`-` は減算）
6. キャンセル時、現在金額は変更しない
7. 現在金額はブラウザの `localStorage` に永続化され、画面リロード後も復元される

## 入力制約

- 入力は正の整数のみ

## 非機能要件

- キーボード操作で閉じられる（Esc 等）
- キーボード操作で確定できる（Enter 等）
- 画面リロード後も現在金額が維持される（`localStorage` から復元される）
